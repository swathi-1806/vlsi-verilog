# vlsi-verilog
## about
This repository contains my Verilog practice codes for VLSI design.
<br>It includes basic to advanced digital circuits.
- combinational
    - adders
    - multiplexer
    - demultiplexer
    - encoder
    - decoder
    - comparators
    - code converters
- sequential circuits
    - latches
    - flipflops
        - sr-ff
        - jk-ff
        - d-ff
        - t-ff
      - FSM
     
### tools used
- model sim
- questa sim
