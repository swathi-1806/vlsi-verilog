# star pattern logic
## square_star_pattern:-

|  position  |-------|-------|--------|------|--------| 
|------------|-------|-------|--------|------|--------|
|            |  j=1  |  j=2  |  j=3   |  j=4   |  j=5 |
|  i=1       |    *  |    *  |    *   |    *   |   *  |
|  i=2       |    *  |    *  |    *   |    *   |   *  |
|  i=3       |    *  |    *  |    *   |    *   |   *  |
|  i=4       |    *  |    *  |    *   |    *   |   *  |
|  i=5       |    *  |    *  |    *   |    *   |   *  | 

```
for(i=0;i<=5;i=i+1)begin
  for(j=0;j<=5;j=j+1)begin
    $write("*");
  end
  $write("\n");
end
```
